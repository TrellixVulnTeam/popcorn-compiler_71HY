352.nab Approved Portability Flags

This is the complete set of approved portability flags for 352.nab.

Portability Flags

  • -DSPEC_NO_SYS_ERRNO_H
    • -DSPEC_NO_SYS_ERRNO_H
    • (?:^|(?<=\s))[-/]DSPEC_NO_SYS_ERRNO_H(?:=\S*)?(?=\s|$)

    • Set if you need to deal with alternate location of errno.h include.

Mandatory Flags

  • -DNOPERFLIB
  • -DNOREDUCE
    • -DNOPERFLIB
    • (?:^|(?<=\s))[-/]DNOPERFLIB(?:=\S*)?(?=\s|$)

    • Tells benchmark not to use perflib. Must be set for SPEC run and is set in object.pm.

    • -DNOREDUCE
    • (?:^|(?<=\s))[-/]DNOREDUCE(?:=\S*)?(?=\s|$)

    • Setting this macro will cause vector reduction with OpenMP to be single- threaded (as it is when using MPI and ScaLAPACK). Set in object.pm for valid SPEC run.

    • -DDATABASE_H
    • (?:^|(?<=\s))[-/]DDATABASE_H(?:=\S*)?(?=\s|$)

    • Guard flag for include file database.h, do not change.

    • -DDOCK
    • (?:^|(?<=\s))[-/]DDOCK(?:=\S*)?(?=\s|$)

    • Guard around include of ssf2.c file, do not change.

    • -DEGB_OMP_FLUSH
    • (?:^|(?<=\s))[-/]DEGB_OMP_FLUSH(?:=\S*)?(?=\s|$)

    • Used for program synchronization, not allowed to be changed.

    • -DFALSE
    • (?:^|(?<=\s))[-/]DFALSE(?:=\S*)?(?=\s|$)

    • Sets the value of false, do not change.

    • -DHEAP_SORT_PAIRS
    • (?:^|(?<=\s))[-/]DHEAP_SORT_PAIRS(?:=\S*)?(?=\s|$)

    • Tells the code to use heapsort_pairs, do not change.

    • -DLINT
    • (?:^|(?<=\s))[-/]DLINT(?:=\S*)?(?=\s|$)

    • Setting this value removes code, not allowed.

    • -DLOGARITHMIC_REDUCTION
    • (?:^|(?<=\s))[-/]DLOGARITHMIC_REDUCTION(?:=\S*)?(?=\s|$)

    • Tells the code to use logarithmic reduction, not allowed to be set.

    • -DMMCIF
    • (?:^|(?<=\s))[-/]DMMCIF(?:=\S*)?(?=\s|$)

    • Guard around include of cifparse.h, do not change.

    • -DMME_LOGARITHMIC_REDUCTION
    • (?:^|(?<=\s))[-/]DMME_LOGARITHMIC_REDUCTION(?:=\S*)?(?=\s|$)

    • Tells the code to use logarithmic reduction, not allowed to be set.

    • -DMME_OMP_FLUSH
    • (?:^|(?<=\s))[-/]DMME_OMP_FLUSH(?:=\S*)?(?=\s|$)

    • Used for program synchronization, not allowed to be changed.

    • -DMORT
    • (?:^|(?<=\s))[-/]DMORT(?:=\S*)?(?=\s|$)

    • Alternate definition of function, not allowed to be changed.

    • -DMPI
    • (?:^|(?<=\s))[-/]DMPI(?:=\S*)?(?=\s|$)

    • This is to use with MPI, not allowed.

    • -DNAB_DOUBLE_PRECISION
    • (?:^|(?<=\s))[-/]DNAB_DOUBLE_PRECISION(?:=\S*)?(?=\s|$)

    • Used for double precision version of benchmark. Must not be set.

    • -DNOPAR
    • (?:^|(?<=\s))[-/]DNOPAR(?:=\S*)?(?=\s|$)

    • Causes parallel execution to be avoided, must not be set.

    • -UNOPERFLIB
    • (?:^|(?<=\s))[-/]UNOPERFLIB(?:=\S*)?(?=\s|$)

    • Tells benchmark not to use perflib. Must not be unset for SPEC run.

    • -UNOREDUCE
    • (?:^|(?<=\s))[-/]UNOREDUCE(?:=\S*)?(?=\s|$)

    • Setting this macro will cause vector reduction with OpenMP to be single- threaded (as it is when using MPI and ScaLAPACK). NOREDUCE is required for valid SPEC run, do not unset.

    • -DOPENMP
    • (?:^|(?<=\s))[-/]DOPENMP(?:=\S*)?(?=\s|$)

    • We use SPEC_OMP, which is set by default, so don't use OPENMP for valid SPEC run.

    • -DPI
    • (?:^|(?<=\s))[-/]DPI(?:=\S*)?(?=\s|$)

    • Sets the value of PI in the code, do not change.

    • -DPRINT_14PAIRS
    • (?:^|(?<=\s))[-/]DPRINT_14PAIRS(?:=\S*)?(?=\s|$)

    • Debug printing of data, do not set for SPEC runs.

    • -DPRINT_DERIV
    • (?:^|(?<=\s))[-/]DPRINT_DERIV(?:=\S*)?(?=\s|$)

    • This is a debug print controller, not to be used for nomral runs.

    • -DPRINT_EPHI
    • (?:^|(?<=\s))[-/]DPRINT_EPHI(?:=\S*)?(?=\s|$)

    • This is a debug print controller, not to be used for normal runs.

    • -DSCALAPACK
    • (?:^|(?<=\s))[-/]DSCALAPACK(?:=\S*)?(?=\s|$)

    • This is to use with SCALAPACK, not allowed.

    • -DSORT_ATOM_NUMBERS
    • (?:^|(?<=\s))[-/]DSORT_ATOM_NUMBERS(?:=\S*)?(?=\s|$)

    • Tells the code how to sort atoms, must not be changed for SPEC runs.

    • -DSPEC_DEBUG
    • (?:^|(?<=\s))[-/]DSPEC_DEBUG(?:=\S*)?(?=\s|$)

    • Allows for SPEC debug information, not for valid runs.

    • -DTRUE
    • (?:^|(?<=\s))[-/]DTRUE(?:=\S*)?(?=\s|$)

    • Sets the value of true, do not change.

    • -D__NABCODE_H__
    • (?:^|(?<=\s))[-/]D__NABCODE_H__(?:=\S*)?(?=\s|$)

    • Guard for nabcode.h include file, do not change.

    • -D__NABTYPES_H__
    • (?:^|(?<=\s))[-/]D__NABTYPES_H__(?:=\S*)?(?=\s|$)

    • Guard for nabtypes.h include file, do not change.

    • -D__NAB__
    • (?:^|(?<=\s))[-/]D__NAB__(?:=\S*)?(?=\s|$)

    • Guard for nab.h include file, do not change.

    • -D__cplusplus
    • (?:^|(?<=\s))[-/]D__cplusplus(?:=\S*)?(?=\s|$)

    • System define of C++ syntax rules needed.

    • -D_regexp_h
    • (?:^|(?<=\s))[-/]D_regexp_h(?:=\S*)?(?=\s|$)

    • Guard around inline include file of defines in regexp.c file, do not change.
